# VirusTotal MCP Server

This MCP server provides functionality avilable from VT API. You need to obtain a key from VT first in order to run this example. 
Note that, in the free tier, VT allows you to do only 4 API queries per minute. 

```
VT_API_KEY=<vt-api-key>
```

## Functionalities supported

- Scan a URL
- Fetch the VT report for a URL

## Running the agent
```
adk web
```
