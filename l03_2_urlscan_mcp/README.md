# URLScan MCP Server

This mcp server offers functionality available from URLScan API. You need to get an API key from urlscan.io and set it to the following environmental variable.

```
URLSCAN_API_KEY=<urlscan-io-api-key>
```

## Functionalities supported
- Fetch URL Scan crawling report
- Fetch screenshot
- Fetch DOM

## Running the agent
```
adk web
```
